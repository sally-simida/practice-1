差异化选择，一个或多个 ，类选择器 class

class 的特殊用法，在一个标签内部，只能有一个class，用 . 调用； 一个标签可以设置多个类名，用空格隔开，可以进行多种选择！最常用！

id 选择器：元素的id是唯一的，但是类名是可以多次使用的 用 # 调用！ 和js搭配使用！

通配符选择器： * 调用 ；选择所有的标签！ 如： * { margin: 0 ;//定义外边距 padding: 0;//定义内边距 } 会匹配页面所有的元素，降低页面响应速度，不建议使用！



字体样式

字体大小：font-size : 16px ；； 字体：fout-family:"某字体";//可以写多个字体，用‘，’隔开，（某些浏览器不支持的时候，可有多个选择，如都没有使用电脑默认字体）；； 字体粗细：font-weight: 700;(700=bold) 700 不能跟单位；； font-weight: 400;(400=normal);; 字体风格：font-style: italic;//字体倾斜 font-style:normal;;//让字体不倾斜；；； font综合写法 font: italic 700 20px "楷体";//顺序不能更改，后两个不能省略；；；
